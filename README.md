📊 Food Product Health Analysis & Prediction
🔹 Overview

This project analyzes food product data using the OpenFoodFacts API to identify health patterns and build a machine learning model to classify products as healthy or unhealthy.

🔹 Dataset
Source: OpenFoodFacts API
Records: 1000+ food products
Features: Brand, Category, Nutrition values, NutriScore
🔹 Tech Stack
Python (Pandas, NumPy)
Data Visualization (Matplotlib, Seaborn)
Machine Learning (Scikit-learn)
Power BI (Dashboard)
🔹 Key Insights
~65–70% products are unhealthy
Sugar and fat strongly impact health score
Few brands dominate product distribution
🔹 Model
Algorithm: Random Forest
Accuracy: ~85%
